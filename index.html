How to Use

Load a quiz file in one of the supported formats:

  • JSON Format: Structured data with questions, options, and explanations
  • XML Format: Well-formed XML with quiz structure
  • TXT Format: Simple text format with specific delimiters

View Format Examples

JSON Format Example:

[
  {
    "id": 1,
    "title": "Question Title",
    "question": "What is your question?",
    "options": [
      "A) Option 1",
      "B) Option 2", 
      "C) Option 3",
      "D) Option 4"
    ],
    "correct_answer": "A",
    "explanation": "Detailed explanation here..."
  }
]

XML Format Example:

<quiz>
  <question id="1">
    <title>Question Title</title>
    <text>What is your question?</text>
    <options>
      <option key="A">Option 1</option>
      <option key="B">Option 2</option>
    </options>
    <correct>A</correct>
    <explanation>Detailed explanation here...</explanation>
  </question>
</quiz>
